# Login con ID Uruguay

Este proyecto es un **componente Jakarta EE** que implementa el flujo completo de **Login y Logout con ID Uruguay (GUB.UY)** utilizando OpenID Connect.

## Descripción

Pequeño componente que demuestra cómo integrar el inicio y cierre de sesión con **ID Uruguay**, incluyendo el manejo del `id_token`, redirecciones, y la finalización correcta de la sesión tanto en la aplicación como en el proveedor de identidad.

## Tecnologías utilizadas

- Java / Jakarta EE  
- Servlets  
- OpenID Connect  
- WildFly  
- ID Uruguay (GUB.UY)
